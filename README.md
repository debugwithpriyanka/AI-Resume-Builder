# AI Resume Builder

An AI-powered Resume Builder developed using CrewAI, Ollama, FastAPI and React.

## Features

- ATS Friendly Resume
- ATS Report
- Career Improvement Plan
- Cover Letter
- LinkedIn About
- Interview Questions
- AI Job Search
- Dark Mode

## Tech Stack

- Python 3.11
- CrewAI
- Ollama
- FastAPI
- React
- Vite

## Run Backend

cd backend

uvicorn main:app --reload

## Run Frontend

cd resume-builder-ui

npm install

npm run dev
